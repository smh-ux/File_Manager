# File_Manager

File_Manager is a file manager (for windows) based on python. It was made by [Semih Okumu≈ü](https://github.com/smh-ux) as a personal project. With this Python-based file manager, you can copy, move, rename and delete your files to the target directory.

## Modules Used:

1. tkinter
2. os
3. shutil

## Functions you can use:

1. Select file
2. Delete file
3. Copy file
4. Move file
5. Rename file
